# Memory: YouTube Commands

- **YouTube Processing**: When a YouTube link is received, call the `notebooklm-python` tool/program.
- **Commands**:
    - `[URL] /nb-s`: Generate a **Presentation** (簡報) using `notebooklm-python`.
    - `[URL] /nb-v`: Generate a **Video** (影片) using `notebooklm-python`.
- **Language**: All outputs must be in **Traditional Chinese (繁體中文)**.
- **Reference**: https://github.com/teng-lin/notebooklm-py
