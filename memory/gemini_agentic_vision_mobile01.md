## 🎯 什麼是 Agentic Vision？

**定義**：讓 AI 模型仿人類的**視覺推理 → 規劃 → 行動**的日常行為

**核心特色**：
- 給模型**一張圖片 + 問題**
- 模型自動展開**自主性視覺推理**
- **全程無須人為干預**

## ⚠️ 注意事項

### 限制與風險
1. **隱私問題**：上傳敏感圖片到 Google 雲端
2. **準確度**：雖然水準高，但仍需人工驗證
3. **成本**：Gemini API 調用費用

### 給客戶的建議
> 「Agentic Vision 雖然強大，但涉及敏感資訊的圖片，
> 建議使用本地部署的 OpenClaw + 本地視覺模型處理，
> 確保數據不會上傳到外部雲端。」
