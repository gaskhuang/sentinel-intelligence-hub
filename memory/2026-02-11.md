## 🤖 新任務：雙龍蝦 Agent 系統 (Side Project 自動化工廠)

### 需求確認
G大要求建立**雙 Agent 協作系統**，使用 auto-skill 模組：

**架構設計：**
1. **Main Agent（大腦）**：回顧聊天記錄 → 構思 Side Project → 產出需求文件
2. **Coding Agent（執行）**：接收需求 → 實作程式碼 → 推上 GitHub

**自動化流程（每天凌晨 3 點）：**
1. Main Agent 回顧前一天的聊天記錄
2. 根據對 G大的了解與興趣，構思有趣的 Side Project
3. 產出技術需求文件（PRD）
4. Coding Agent 接收 PRD 並實作
5. 完成後自動推送到 GitHub

**技術規劃：**
- 使用 `sessions_spawn` 建立獨立的 Coding Agent session
- 使用 `cron` 設定每日凌晨 3 點觸發
- Main Agent 與 Coding Agent 透過檔案或 session 傳遞需求
- 使用 `auto-skill` 建立經驗累積機制
