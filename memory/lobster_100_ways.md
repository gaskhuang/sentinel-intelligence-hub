# 🦞 龍蝦一百種用法 (Lobster 100 Ways) - 商業變現戰略全清單

## 核心目標：以「抓取、分析、自動化」為核心，打造商務人士必備的數位龍蝦。

### 第一階段：情報監控與社群採集 (Intelligence & Social Scraping)
1.  [ ] **龍蝦 X (Twitter) 輿情監控**：24/7 監控關鍵字，自動彙整市場風向報告。
2.  [ ] **龍蝦 Threads 爆文抓取**：精準分析競爭對手流量密碼，自動產出文案靈感。
3.  [ ] **龍蝦 Facebook 社團採集**：自動提取潛在客戶需求，彙整至 Google Sheets。
4.  [ ] **龍蝦 IG/小紅書 趨勢分析**：抓取視覺數據，預測下一個商業爆品。
5.  [ ] **龍蝦 LinkedIn 獵才獵人**：自動根據職缺關鍵字，篩選並整理人才清單。
6.  [ ] **龍蝦 104 產業人力動態**：監控特定產業的徵才頻率，判斷市場擴張訊號。
7.  [ ] **龍蝦 Reddit 社群情緒監控**：監控國外論壇對特定技術或產品的真實評價。
8.  [ ] **龍蝦 YouTube 競品評論抓取**：分析對手影片下方的用戶痛點，尋找產品突破口。
9.  [ ] **龍蝦 App Store/Google Play 評論分析**：自動抓取並歸納用戶對 App 的功能需求與不滿。
10. [ ] **龍蝦 專業論壇/報紙 標題監控**：即時掌握產業最新論文或報導，保持情報領先。

### 第二階段：深度分析與產出 (Deep Analysis & Output)
11. [ ] **龍蝦 YouTube 轉商業簡報**：一鍵將長影片提煉成 10 頁執行摘要 PPT。
12. [ ] **龍蝦 Podcast 筆記本**：抓取音訊內容，自動產出繁體中文逐字稿與關鍵決策點。
13. [ ] **龍蝦 多文件財報對比**：同時讀取 5 份 PDF，分析年度增長率與風險因子。
14. [ ] **龍蝦 Web 搜尋情報網**：全網搜尋（含 Reddit/Facebook），產出精美 HTML 情報報告。
15. [ ] **龍蝦 競爭對手網站監控**：只要對方官網一更新，龍蝦立刻發通知並分析策略。
16. [ ] **龍蝦 龍蝦精華協議 (Distilled Asset)**：自動壓縮萬字會議記錄，提煉成 300 字執行方針。
17. [ ] **龍蝦 技術白皮書深讀**：將枯燥的技術文件轉化為易懂的商業決策建議。
18. [ ] **龍蝦 品牌提及交叉分析**：分析品牌在不同平台（X, FB, Threads）的聲量與情緒對比。
19. [ ] **龍蝦 病毒式內容解構**：分析爆紅內容的結構、節奏與鉤子，產出可複製模板。
20. [ ] **龍蝦 法律條款異動摘要**：自動比對新舊版服務條款，指出對我方不利的變更。
